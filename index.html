The Seven RESTful Routes

A common pattern to satisfy the CRUD operations in your app using seven RESTful routes.

Learn more
Name Path HTTP Verb Purpose Mongoose Method
Index /workers GET List all workers Worker.find()
New /workers/new GET Show new worker form N/A (No database call)
Create /workers POST Add a new worker, then redirect somewhere Worker.create()
Show /workers/:id GET Show info about one specific worker Worker.findById()
Edit /workers/:id/edit GET Show edit form for one worker Worker.findById()
Update /workers/:id PUT Update a particular worker, then redirect somewhere Worker.findByIdAndUpdate()
Delete /workers/:id DELETE Delete a particular worker, then redirect somewhere Worker.findByIDAndRemove()
